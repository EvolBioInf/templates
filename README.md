# `templates`
## Author
[Bernhard Haubold](http://thymine.evolbio.mpg.de/), `haubold@evolbio.mpg.de`
## Description
Generate code templates.
## Compilation
Compile the sources `make` and find the executables `makeCproject` and
`makeCfile` in the
directory `build`.
## Documentation
The command
`make doc`
generates the manual `doc/templates.pdf`.
